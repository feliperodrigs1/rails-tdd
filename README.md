# rails-tdd
Udemy course, focused on test-driven development (TDD), using Rails in conjunction with RSpec, Capivara and other Gems.

<b>Was used:</b>
<br> - Ruby 2.7.8
<br> - Rails 5.1.7
<br> Due to the date of the course

<b>Some Gems learned:</b>
- RSpec
- Capybara
- Devise
- Responders
- factory_bot
- Rubocop
- faker
- HTTParty
- WebMock
- VCR
- Shoulda Matchers
- JsonExpectations
- JsonMatchers
- spring-commands
